# Documentação Importação VS 2026 sem EDMX

Este pacote é um site estático simples.

## Como subir na Vercel

1. Extraia o ZIP.
2. Suba a pasta para um repositório GitHub.
3. Na Vercel, importe esse repositório.
4. Framework Preset: Other.
5. Build Command: deixe vazio.
6. Output Directory: deixe vazio.
7. Deploy.

O arquivo principal é `index.html`, na raiz.
